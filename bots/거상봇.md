# 거상봇 (v2.3)

거상 온라인 연동 — 사통팔달 자동 알림, 육의전 아이템 검색 및 알람, 메모, 퀘스트 정보 조회

## 기능

- **사통팔달** — 대상 채팅방에서 채팅 감지 시 서버에서 신규 데이터를 가져와 자동 전송
  - 5분 쿨다운으로 과도한 요청 방지
- **육의전** — 서버 DB에서 아이템 검색
- **알람 시스템** — 유저별 알람 등록/해제/조회. 유저별 그룹핑하여 표시
- **메모** — 유저별 메모 저장/조회/삭제 (로컬 JSON)
- **퀘스트 정보** — `gersangjjang.com` 파싱 결과를 로컬 JSON 번들에서 조회 (네트워크 호출 없음)

## 명령어

| 명령어 | 설명 | 권한 |
|--------|------|------|
| `!육의전 <이름>` | DB에서 아이템 검색 | 전체 |
| `!알람등록 <이름>` | 신규 아이템 알람 등록 (유저별) | 전체 |
| `!알람해제 <이름>` | 본인의 해당 키워드 알람 해제 | 전체 |
| `!알람해제 <번호>` | 해당 번호 유저의 알람 전체 해제 | 전체 |
| `!알람목록` | 유저별 그룹핑된 알람 리스트 | 전체 |
| `!메모 <내용>` | 메모 저장 | 전체 |
| `!메모` | 본인 메모 목록 조회 | 전체 |
| `!메모삭제 <번호>` | 해당 번호 메모 삭제 | 전체 |
| `!퀘스트` | 카테고리 요약 및 사용법 | 전체 |
| `!퀘스트 목록` | 전체 퀘스트 이름 (카테고리별) | 전체 |
| `!퀘스트 목록 <카테고리>` | 해당 카테고리의 퀘스트 이름 | 전체 |
| `!퀘스트 주간` | `주간-일반` (/quest/week.asp) | 전체 |
| `!퀘스트 일일` | `일일-우호도` (/quest/date2.asp) | 전체 |
| `!퀘스트 <이름>` | 퀘스트 상세 (부분 일치 시 후보 목록) | 전체 |

> 사통팔달은 대상 채팅방에서 채팅이 발생하면 자동으로 동작합니다.

## 퀘스트 데이터 번들

- **경로**: `sdcard/msgbot/Bots/거상봇/quests.json`
- **생성**: `python3 server/scripts/parse_gersangjjang.py` (저장소에서 수동 실행)
- **출처**: `https://www.gersangjjang.com/quest/index.asp` (전체 서브카테고리)
- **제외**: `일일-일반` (/quest/date.asp) — 사용자 지시
- **카테고리 분류 (1차)**: 진행(11) · 지역(45) · 주간일일(4) · 전장(8) · 이벤트(6) · 서비스(4) = 78개

## 설정

| 상수 | 설명 |
|------|------|
| `FASTAPI_BASE_URL` | FastAPI 서버 주소 |
| `SATONG_COOLDOWN_MS` | 사통팔달 쿨다운 (5분) |
| `YUK_NOTIFY_COOLDOWN_MS` | 육의전 알림 쿨다운 (5분) |
| `TARGET_ROOM_IDS` | 대상 채팅방 ID 리스트 |
| `QUEST_FILE` | 퀘스트 번들 JSON 경로 |
