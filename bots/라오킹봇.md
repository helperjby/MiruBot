# 라오킹봇 (v1.7)

라이즈 오브 킹덤즈 일정 관리 — 주기성 이벤트, 브리핑, 생일 알림

## 기능

- **주기성 이벤트** — 폐허/제단 등 반복 이벤트의 다음 일정 계산 및 안내
  - 패배/예외 기간 처리 지원
- **일일 브리핑** — 매일 09:00 자동 공지, 오늘/내일 일정 조회
- **연대기** — 전체 남은 일정 한눈에 조회, 시간 조정 기능
- **일정 관리** — 일회성 일정 등록/삭제
- **생일 관리** — 생일 등록/제거, 당일 자동 축하
- **공지 전파** — 공지방 → 수다방 자동 전파
- **이미지 전송** — 동맹 구도, 사령관 특성 이미지

## 명령어

### 주기성 이벤트

| 명령어 | 설명 | 권한 |
|--------|------|------|
| `!폐허` | 다음 폐허 일정 | 전체 |
| `!제단` | 다음 제단 일정 | 전체 |

### 브리핑

| 명령어 | 설명 | 권한 |
|--------|------|------|
| `!오늘` | 오늘의 일정 브리핑 | 전체 |
| `!내일` | 내일의 일정 브리핑 | 전체 |
| `!연대기` | 남은 연대기 일정 | 전체 |
| `!연대기 시간조정 <+-hhmm>` | 연대기 시간 조정 | 전체 |

### 일정/생일 관리

| 명령어 | 설명 | 권한 |
|--------|------|------|
| `!일정등록` | 일회성 일정 등록 | 전체 |
| `!일정삭제` | 일정 삭제 | 전체 |
| `!일정` | 등록된 일정 목록 | 전체 |
| `!생일등록 <닉네임>, <MMDD>` | 생일 등록 | 전체 |
| `!생일제거 <닉네임>` | 생일 제거 | 전체 |
| `!생일목록` | 등록된 생일 목록 | 전체 |

### 기타

| 명령어 | 설명 | 권한 |
|--------|------|------|
| `!크븝 동맹구도` | 동맹 구도 이미지 전송 | 전체 |
| `!특성 <사령관이름>` | 사령관 특성 이미지 전송 | 전체 |
| `!명령어` | 명령어 목록 | 전체 |

## 설정

| 상수 | 설명 |
|------|------|
| `ALLOWED_COMMAND_ROOM_IDS` | 명령어 허용 방 ID |
| `KVK_CONTEXT` | 현재 KVK 시즌 정보 |
| `EVENTS_CONFIG` | 주기성 이벤트 설정 (기준시간, 주기, 종료시간) |
| `DAILY_ANNOUNCE_CONFIG` | 일일 자동 공지 설정 (방, 시간) |
| `ONE_TIME_EVENTS_PATH` | 일회성 일정 데이터 경로 |
| `BIRTHDAY_EVENTS_PATH` | 생일 데이터 경로 |
