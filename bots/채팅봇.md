# 채팅봇 (v1.5.0)

커뮤니티 시스템 — 레벨, 경험치, 칭호, 추천/비추천, 닉네임 히스토리

## 기능

- **경험치/레벨** — 채팅 참여 시 XP 획득, 자동 레벨업
- **칭호 시스템** — 레벨 구간별 7단계 칭호 (새싹 → 신화)
- **추천/비추천** — 하루 1회 제한, 유저 간 평판 시스템
- **월간/누적 랭킹** — 월간 XP 자동 리셋, 누적 TOP15 + 월간 TOP15
- **닉네임 히스토리** — 채팅방별 닉네임 변경 이력 추적
- **방별 활성화** — 봇 적용 채팅방 관리

## 명령어

| 명령어 | 설명 | 권한 |
|--------|------|------|
| `!정보 [닉네임]` | 유저 프로필 (레벨, XP, 칭호, 추천수) | 전체 |
| `!추천 <닉네임>` | 유저 추천 (하루 1회) | 전체 |
| `!비추 <닉네임>` | 유저 비추천 (하루 1회) | 전체 |
| `!랭킹채팅` | 누적 TOP15 + 월간 TOP15 | 전체 |
| `!정보입력` | 유저 정보 수동 입력 | 전체 |
| `!추천알람` | 추천 알림 ON/OFF 토글 | 전체 |
| `!커뮤니티` | 커뮤니티 기능 ON/OFF (정보/추천/비추 토글) | 관리자 |
| `!활성화` | 현재 방에서 봇 활성화 | 관리자 |
| `!비활성화` | 현재 방에서 봇 비활성화 | 관리자 |

## 설정

| 상수 | 설명 |
|------|------|
| `MAX_UPVOTES_PER_DAY` | 하루 추천 횟수 제한 (기본 1) |
| `MAX_DOWNVOTES_PER_DAY` | 하루 비추천 횟수 제한 (기본 1) |
| `CONFIG_PATH` | 활성화 방 목록 경로 (`active_rooms.json`) |
| `COMM_SETTINGS_FILE` | 커뮤니티 설정 경로 (`comm_settings.json`) |
