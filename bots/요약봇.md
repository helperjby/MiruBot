# 요약봇 (v1.4.0)

채팅 로그 수집, Gemini AI 요약, 유저별 통계 및 인물평 분석

## 기능

- **채팅 로그 수집** — 인메모리 버퍼링 후 서버로 배치 전송
  - 50개 도달 또는 60초 경과 시 자동 flush
- **대화 요약** — 최근 N시간 대화를 Gemini AI로 요약
- **채팅 통계** — 유저별 시간대/요일별 채팅 통계
- **인물평** — LLM 기반 유저 성격 분석
- **나이 추정** — 층화 샘플링 기반 LLM 나이 추정
- **방별 DB 토글** — 채팅방별 로그 기록 ON/OFF

## 명령어

| 명령어 | 설명 | 권한 |
|--------|------|------|
| `!요약` | 최근 4시간 대화 요약 | 전체 |
| `!요약 <N>` | 최근 N시간 요약 (1~12) | 전체 |
| `!채팅통계 <닉네임>` | 유저별 채팅 통계 (시간대/요일별) | 전체 |
| `!인물평 <닉네임>` | LLM 인물평 분석 | 전체 |
| `!나이 <닉네임>` | LLM 나이 추정 | 전체 |
| `!DB` | 현재 방 채팅 기록 ON/OFF 토글 (기본: OFF) | 전체 |

## 설정

| 상수 | 설명 |
|------|------|
| `FASTAPI_BASE_URL` | FastAPI 서버 주소 |
| `BATCH_SIZE` | 배치 flush 임계값 (50개) |
| `FLUSH_INTERVAL_MS` | 타이머 flush 간격 (60초) |
| `DEFAULT_SUMMARY_HOURS` | 기본 요약 시간 (4시간) |
| `MAX_SUMMARY_HOURS` | 최대 요약 시간 (12시간) |
| `DB_ROOMS_PATH` | DB 활성화 방 목록 경로 |
