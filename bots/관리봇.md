# 관리봇 (v1.2.0)

봇 관리, 모니터링, 유저 차단 등 오너 전용 관리 유틸리티

## 기능

- **전원 제어** — 개별 봇 켜기/끄기
- **원격 컴파일** — 봇 스크립트 원격 컴파일
- **로그 조회** — 봇별 최근 로그 조회
- **상태 모니터링** — 업타임, 메모리, 봇 상태 확인
- **방별 적용 관리** — 봇을 특정 채팅방에만 적용/제거
- **오너 해시 관리** — 하드코딩 + 파일 기반 병합 관리
- **유저 차단** — 크로스봇 공유 파일 기반 차단 시스템

## 명령어

### 공개 명령어

| 명령어 | 설명 |
|--------|------|
| `!핑` / `!테스트` | 응답 확인 |
| `!여기` / `!방정보` | 현재 채팅방 정보 |
| `!나` / `!내정보` | 유저 해시 정보 |
| `!관리도움` / `!관리명령어` | 명령어 목록 |

### 오너 전용 명령어

| 명령어 | 설명 |
|--------|------|
| `!봇목록` | 등록된 봇 목록 |
| `!응답체크` | 전체 봇 응답 확인 |
| `!봇켜기 <봇>` | 봇 활성화 |
| `!봇끄기 <봇>` | 봇 비활성화 |
| `!컴파일 <봇>` | 봇 원격 컴파일 |
| `!로그 [봇] [N]` | 봇 로그 조회 (기본 10건, 최대 30건) |
| `!업타임` | 관리봇 업타임 |
| `!메모리` | 메모리 사용량 |
| `!봇상태` | 전체 봇 상태 요약 |
| `!적용설정` | 방별 봇 적용 관리 |
| `!오너등록` | 현재 해시를 오너로 등록 |
| `!오너해시추가 <해시>` | 특정 해시를 오너로 추가 |
| `!해시검색 <키워드>` | 오너 해시 검색 |
| `!권한테스트` | 오너 권한 확인 |
| `!객체확인` / `!덤프` | 객체 디버그 덤프 |
| `!차단 <해시>` | 유저 차단 (크로스봇) |
| `!차단해제 <해시>` | 유저 차단 해제 |
| `!차단목록` | 차단 유저 목록 |

## 설정

| 상수 | 설명 |
|------|------|
| `OWNER_HASHES_DEFAULT` | 하드코딩 오너 해시 목록 |
| `OWNER_HASHES_PATH` | 파일 기반 오너 해시 경로 |
| `BOT_ROOMS_PATH` | 방별 봇 적용 설정 경로 |
| `BLOCKED_USERS_PATH` | 공유 차단 유저 파일 경로 |
