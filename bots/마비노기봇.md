# 마비노기봇 (v2.8.1)

마비노기 모바일 게임 연동 — 실시간 알림, 룬워드 사전, 열쇠 운세

## 기능

- **어비스/심층 구멍 알림** — 앱 알림 감지 시 지정된 채팅방에 실시간 전파
  - NPC별 알림 대상 방 설정 (`ALERT_ROOM_MAP`)
  - 쿨타임 관리로 중복 알림 방지
  - 기능별 ON/OFF 토글 (`!어구알람`, `!심구알람`)
- **공지사항 알림** — 점검/업데이트 공지만 필터링하여 전파
  - ON/OFF 토글 (`!공지알람`)
- **룬워드 사전** — 룬 정보 검색, 룬워드 조합 조회, 룬 비교
- **초월** — 초월 정보 조회
- **열쇠 운세** — 마비노기 열쇠 운세 미니게임
- **말티즈 자동 알림** — 말티즈 캐릭터 알림 자동 전송

## 명령어

| 명령어 | 설명 | 권한 |
|--------|------|------|
| `!명령어` | 명령어 목록 | 전체 |
| `!운세` / `!열쇠운세` | 열쇠 운세 미니게임 | 전체 |
| `!말티즈` | 말티즈 정보 조회 | 전체 |
| `!룬워드 <검색어>` | 룬워드 조합 검색 | 전체 |
| `!룬정보 <룬이름>` | 개별 룬 상세 정보 | 전체 |
| `!룬비교 <룬1> <룬2>` | 룬 스탯 비교 | 전체 |
| `!초월` | 초월 정보 조회 | 전체 |
| `!알람리스트` / `!알람목록` | 알림 ON/OFF 상태 확인 | 전체 |
| `!어구알람` | 어비스 구멍 알림 ON/OFF | 관리자 |
| `!심구알람` | 심층 구멍 알림 ON/OFF | 관리자 |
| `!공지알람` | 공지사항 알림 ON/OFF | 관리자 |

## 설정

| 상수 | 설명 |
|------|------|
| `FASTAPI_BASE_URL` | FastAPI 서버 주소 |
| `DB_PATH` | mabi_config.json 경로 |
| `RUNE_INFO_JSON_PATH` | 룬 정보 JSON 경로 |
| `RUNEWORD_JSON_PATH` | 룬워드 JSON 경로 |
| `ALLOWED_ROOM_NAMES` | 명령어 허용 방 이름 목록 |
| `ALERT_ROOM_MAP` | NPC별 알림 대상 방 매핑 |
| `ADMIN_HASHES` | 관리자 해시 목록 |
| `CONFIG_CACHE_TTL` | config 캐싱 TTL (1분) |
