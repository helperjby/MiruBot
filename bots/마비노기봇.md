# 마비노기봇 (v2.8.2)

마비노기 모바일 게임 연동 — 실시간 알림, 룬워드 사전, 열쇠 운세

## 기능

- **어비스/심층 구멍 알림** — 앱 알림 감지 시 지정된 채팅방에 실시간 전파
  - NPC별 알림 대상 방 설정 (`ALERT_ROOM_MAP`)
  - 쿨타임 관리로 중복 알림 방지
  - 기능별 ON/OFF 토글 (`!어구알람`, `!심구알람`)
- **공지사항 알림** — 점검/업데이트 공지만 필터링하여 전파
  - ON/OFF 토글 (`!공지알람`)
- **룬워드 사전** — 룬 정보 검색, 룬워드 조합 조회, 룬 비교
- **초월** — 초월 정보 조회
- **열쇠 운세** — 마비노기 열쇠 운세 미니게임
- **말티즈 자동 알림** — 말티즈 캐릭터 알림 자동 전송

## 트러블슈팅: 이미지 전송 누락 (씹힘 현상)

`MediaSender`로 이미지를 전송할 때 로그에는 '전송 성공'이 찍히지만 실제 카카오톡 방에 이미지가 안 올라오는 현상이 있었음. 원인 두 가지:

### 1. 텍스트-이미지 인텐트 충돌

`bot.send`로 텍스트를 보낸 직후 `sender.send`로 이미지를 전송하면, 안드로이드의 카카오톡 화면 전환 및 공유 인텐트 처리가 겹쳐 이미지 공유 인텐트가 무시됨.

**해결**: 텍스트 전송 후 `java.lang.Thread.sleep(800)` 이상의 딜레이를 두고 이미지 전송.

### 2. URL 쿼리스트링에 의한 확장자 오염

API 응답 URL에 `?alt=media&token=...` 같은 파라미터가 붙어 있을 경우, `lastIndexOf(".")`로 확장자를 추출하면 `.jpg?alt=media` 형태로 저장됨. 안드로이드가 이를 유효한 이미지 파일로 인식하지 못해 전송 누락.

**해결**: `let cleanUrl = imgUrl.split("?")[0];`으로 쿼리스트링을 제거한 뒤 확장자 추출.

> **규칙**: 이미지 전송 코드를 작성할 때는 항상 (1) 텍스트 전송 후 딜레이, (2) URL 파싱 시 쿼리스트링 제거를 적용할 것.
