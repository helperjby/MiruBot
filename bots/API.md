# 외부 연동 봇 (v2.4)

채팅에 공유된 URL을 자동 요약하고, 번역·이미지 전송·환율/증시 조회를 제공하는 봇

## 기능

- **URL 자동 요약** — 채팅에 공유된 링크를 Gemini AI로 요약 (뉴스, YouTube, 커뮤니티 등)
  - YouTube Shorts URL 제외
  - URL 중복 방지 (2초 내 동일 URL 무시)
  - 특정 채팅방 제외 가능 (`URL_SUMMARY_EXCLUSION_LIST`)
- **번역** — 한/영 자동 감지 번역 (`!번역`)
- **이미지 전송** — OneDrive 동기화 이미지를 카카오톡으로 전송 (`!윤호`, `!미루`)
  - 1~4장 범위 제한, 10MB 이미지 크기 제한
  - MediaSender 보안 정책 대응 (로컬 다운로드 후 전송)
- **환율/증시** — 네이버 금융 실시간 데이터 조회

## 명령어

| 명령어 | 설명 | 권한 |
|--------|------|------|
| `!번역 <텍스트>` | 한/영 자동 감지 번역 | 전체 |
| `!환율` | 실시간 환율 조회 | 전체 |
| `!증시` | 실시간 증시 조회 | 전체 |
| `!윤호 <N>` | 윤호 이미지 N장 전송 (1~4) | 전체 |
| `!미루 <N>` | 미루 이미지 N장 전송 (1~4) | 전체 |

> URL이 채팅에 공유되면 명령어 없이 자동으로 요약합니다.

## 설정

| 상수 | 설명 |
|------|------|
| `FASTAPI_BASE_URL` | FastAPI 서버 주소 |
| `MAX_IMAGE_SIZE` | 이미지 최대 크기 (10MB) |
| `URL_SUMMARY_EXCLUSION_LIST` | URL 요약 제외 채팅방 ID 목록 |
| `URL_DEDUP_INTERVAL` | URL 중복 판정 간격 (2초) |
