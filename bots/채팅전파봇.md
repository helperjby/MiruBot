# 채팅전파봇 (v1.0)

채팅방 간 메시지 릴레이 — 소스 방의 메시지를 수다방으로 전파

## 기능

- **메시지 릴레이** — 소스 채팅방의 메시지를 수다방으로 자동 전달
  - 소스 방 닉네임 프리픽스 추가
- **개별 방 토글** — 닉네임으로 특정 소스 방 릴레이 ON/OFF
- **관리자 전용 제어**

## 명령어

| 명령어 | 설명 | 권한 |
|--------|------|------|
| `!수감자` | 전체 릴레이 상태 표시 | 관리자 |
| `!수감자 <닉네임>` | 특정 소스 방 릴레이 ON/OFF 토글 | 관리자 |

## 설정

| 상수 | 설명 |
|------|------|
| `DESTINATION_ROOM_NAME` | 전파 대상 방 이름 (수다방) |
| `SOURCE_ROOMS` | 소스 방 ID → 닉네임 프리픽스 매핑 |
| `ADMIN_HASHES` | 관리자 해시 목록 |
